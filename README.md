### About The Project

The project is to create a personal Backtest module that is able to present the data finding onto streamlit web application.

### Built With

This project is built
* [yfinance](https://pypi.org/project/yfinance/)
* [pandas](https://pandas.pydata.org/)
* [streamlit](https://docs.streamlit.io/en/stable/index.html)

### Prerequisites

* yfinance
  ```py
  $ pip install yfinance --upgrade --no-cache-dir
  ```
* streamlit
  ```py
  $ pip install streamlit
  ```

### Installation

Can run python file directly as __main__ or import as a module

<!-- USAGE EXAMPLES -->
## Usage

if run directly as __main__, you can adjust the parameter in different method directly in the file. Run the file using streamlit command

```py
streamlit run Backtest.py
```

This should automatically redirect you to streamlit web application, but is it does not open you can use ctrl+click the link provided in the terminal
